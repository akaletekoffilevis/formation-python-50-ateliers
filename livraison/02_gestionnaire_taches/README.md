# Projet : Gestionnaire de Tâches

## Objectif
Créer une application de gestion de tâches (todo list) en ligne de commande avec persistance des données.

## Fonctionnalités requises
- Ajouter une tâche avec description
- Lister toutes les tâches
- Marquer une tâche comme terminée
- Supprimer une tâche
- Sauvegarder les tâches dans un fichier (JSON ou texte)
- Charger les tâches au démarrage

## Concepts à appliquer
- Manipulation de fichiers (lecture/écriture)
- Structures de données (listes, dictionnaires)
- Fonctions pour organiser le code
- Gestion d'erreurs basique

## Fichiers attendus
- `gestionnaire_taches.py` : Script principal
- `taches.json` ou `taches.txt` : Fichier de sauvegarde (généré automatiquement)
- `README.md` : Ce fichier

## Extensions possibles
- Catégories de tâches
- Dates d'échéance
- Priorité (haute/moyenne/basse)
- Recherche de tâches
