# Projet : Le Pendu

## Objectif
Créer une version en ligne de commande du jeu classique du pendu.

## Fonctionnalités requises
- L'ordinateur choisit un mot aléatoire à partir d'une liste prédéfinie
- L'utilisateur propose des lettres une par une
- Affichage du mot avec les lettres découvertes (ex: P _ T O N)
- Comptage des erreurs (maximum 6 erreurs avant de perdre)
- Affichage du pendu au fur et à mesure des erreurs
- Détection de victoire (toutes les lettres trouvées)
- Détection de défaite (trop d'erreurs)
- Possibilité de rejouer une nouvelle partie

## Concepts à appliquer
- Chaînes de caractères et manipulation
- Listes et ensembles (pour suivre les lettres proposées)
- Boucles (while pour continuer le jeu)
- Conditions (if/else pour vérifier les lettres)
- Fonctions pour organiser le code
- Optionnel : fichiers pour sauvegarder les scores

## Fichiers attendus
- `pendu.py` : Script principal du jeu
- `mots.py` : Module contenant la liste de mots (optionnel)
- `affichage.py` : Module pour l'affichage du pendu (optionnel)
- `README.md` : Ce fichier

## Extensions possibles
- Niveaux de difficulté (facile/moyen/difficile)
- Thèmes de mots (animaux, pays, langage de programmation, etc.)
- Indicateur visuel du pendu amélioré
- Sauvegarde des scores et statistiques
- Mode deux joueurs
