# Projet : Calculatrice Interactive

## Objectif
Implémenter une calculatrice en ligne de commande qui permet à l'utilisateur de réaliser des opérations mathématiques de base.

## Fonctionnalités requises
- Menu avec choix d'opérations : +, -, *, /
- Saisie de deux nombres par l'utilisateur
- Calcul et affichage du résultat
- Gestion de la division par zéro
- Option pour effectuer plusieurs calculs successivement
- Historique des opérations (optionnel)

## Fichiers attendus
- `calculatrice.py` : Script principal
- `README.md` : Ce fichier

## Critères d'évaluation
- Le programme fonctionne sans erreur
- Gestion appropriée des cas limites (division par zéro)
- Code lisible et commenté
- Respect des conventions PEP 8 (nommage, indentation)
